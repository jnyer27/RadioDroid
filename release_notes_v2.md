## RadioDroid v2.0.0

- **Settings search** — Search radio settings by parameter or value.
- **Radio-specific channel editor** — Spinners and switches for driver-defined options (e.g. bandwidth, groups) on supported radios (e.g. NICFW H3, RT85).
- **Sort Channels by Group** removed — Feature was radio-specific; app is universal.
- **Expanded User Guide** — See attached `userguide.md` or the [online user guide](https://jnyer27.github.io/RadioDroid/) for full documentation.

Install: download `app-release.apk` and install on your Android device (Android 7.0+).
