## RadioDroid v3.1.0

### Highlights

- **Main channel list — radio-specific row** — Driver **`Memory.extra`** lines now show as a **vertical stack** (one line per key) instead of a responsive multi-column grid. Fixes inconsistent or missing columns when `RecyclerView` bound rows before the column container had a measured width (often cured only after rotation). The **channel editor** is unchanged.

### Install

Download **`app-release.apk`** and install on Android 7.0+.
