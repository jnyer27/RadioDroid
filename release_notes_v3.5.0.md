## RadioDroid v3.5.0

### Highlights

- **Dense main channel list** — Each row uses **badges** for power, mode, duplex, and your **Customize main screen** slots (when they do not duplicate those). **Frequency** sits on its own line; **tones** collapse to a single summary when present. **Radio-specific** **Memory.extra** fields appear as **one compact line** (e.g. `Groups: A, G · BW: Wide · BusyLock: Off`): group slots **1–4** merge into **Groups: …**, **None**/empty values are hidden, and boolean-style values read as **On**/**Off**.
- **Documentation** — **User guide** main-screen schematic and text updated to match the new layout.

### Install

Download **`app-release.apk`** and install on Android 7.0+. See also **`userguide.md`** (Markdown) or the PDF **User Guide** asset when present.
