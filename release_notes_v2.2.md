## RadioDroid v2.2.0

### Highlights
- **Radio download / upload feedback** — While **Load from radio** or **Save to radio** runs, the app shows an **indeterminate progress bar** and **rotating status messages**. The CHIRP bridge does not expose byte-level progress; this replaces a legacy “cloning… 0 / 256” indicator that never updated.
- **Documentation** — User guide updated to describe transfer feedback.

### Install
Download **`app-release.apk`** and install on Android 7.0+. See also **`userguide.md`** / **[online user guide](https://jnyer27.github.io/RadioDroid/)** or the PDF **User Guide** asset when present.
