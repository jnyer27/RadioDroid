## RadioDroid v2.1.0

### Highlights
- **BLE-to-serial adapters** — Supports multiple UART GATT profiles: HM-10/TI-style (Baofeng clones), Nordic UART, Microchip/ISSC, NICFW/TD-H3. Filtered BLE scan and safer MTU handling (20-byte fallback) for cheap dongles.
- **Documentation** — README and [online user guide](https://jnyer27.github.io/RadioDroid/) updated from root `userguide.md`.

### Install
Download **`app-release.apk`** and install on Android 7.0+. See also **`userguide.md`** (Markdown) or the PDF **User Guide** asset when present.
