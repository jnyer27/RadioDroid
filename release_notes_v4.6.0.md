## RadioDroid v4.6.0

### Highlights

- **RepeaterBook proximity** — **GMRS** (`prox_result.php`) and **amateur** (`prox2_result.php`) proximity searches return the result list without opening each repeater’s **details** page. **Uplink / downlink tones** (**PL** / **TSQ**) are loaded **only when you tap Import selected**, and **only for the rows you selected**—faster searches and fewer HTTP requests.

### Install

Download **`app-release.apk`** and install on Android 7.0+. See also **`userguide.md`** (Markdown) or the PDF **User Guide** asset when present. The guide is also at **[jnyer27.github.io/RadioDroid](https://jnyer27.github.io/RadioDroid/)**.
